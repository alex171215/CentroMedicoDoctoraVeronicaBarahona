# Workflows

## Available workflows

| Shortcut | Description | File |
|----------|-------------|------|
| `$prepare` | Read all files needed for development context | [prepare.md](./prepare.md) |
| `$design-session <feature-name>` | Run a design workflow for a specific feature | [design-session.md](./design-session.md) |
| `$implementation-session <feature-name>` | Run an implementation workflow for a specific feature | [implementation-session.md](./implementation-session.md) |
| `$deliver` | Execute final delivery workflow checks and summary | [deliver.md](./deliver.md) |
| `$?` | Show help with available workflow shortcuts | [help.md](./help.md) |
