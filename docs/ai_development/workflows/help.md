# Help ($?)

**Shortcut**: `$?`

**Description**: Displays available shortcuts and guidance for using the workflow commands.

## Command

`$?`

## Instructions

List all the workflow shortcuts and their description. DO NOT list detailed instructions.

Take the list of workflows table in `docs/ai_development/workflows/README.md` and display it to the user.

Suggest the user to run `$prepare` if they have not done it before.

Be concise, and use simple language.
